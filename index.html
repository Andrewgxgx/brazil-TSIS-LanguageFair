Project Manifest

The Samba Track Music player was engineered as a fast delivery pipeline for traditional Brazilian carnival rhythms. By leveraging Cloudflare's edge network, this platform bypasses traditional egress bottlenecks to serve pure acoustic recordings and raw percussion tracks instantly.
This project is also open sourced on Github! although... there's no proper documentation so just FAFO.

Furthermore, the Samba Drum Machine is fully client side! Built with pure Web Audio API and geeky Math Stuff. This codebase can be found in the github repo as well.
